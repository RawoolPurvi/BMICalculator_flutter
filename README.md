⚖️ BMI Calculator

A Body Mass Index calculator application built using Flutter and Dart that allows users to calculate their BMI based on height and weight inputs. The app provides instant results along with health category feedback such as underweight, normal, overweight, or obese.

✨ Features
Input height and weight to calculate BMI
Instant BMI calculation and result display
Health category interpretation
Clean and responsive UI following Material Design
📦 Packages Used
font_awesome_flutter – Icons for UI elements
cupertino_icons – iOS style icons
flutter material – UI components
